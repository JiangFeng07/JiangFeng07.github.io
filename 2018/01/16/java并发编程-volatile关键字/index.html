java并发编程-volatile关键字

发表于 2018-01-16 | 阅读次数

  • 可见性

          使用volatile来修饰共享变量时，就保证了不同线程对这个变量操作的可见性，也就是说，一个线程如果修改了这个变量的值，会及时把改变后的值更新到主内存中，对其它线程来说是可见的。

  • 有序性

          由于volatile关键字能禁止指令重排序，所以保证了“一定”的有序性。

  • 原子性
